## 项目概述


- Ant  Design+React+Redux实现前端页面


- Node.js+Express+Mysql实现后端服务

- 路由功能使用的是React-Router


由于是学习React的第一个练手项目，所以更多的考虑到功能的实现方面，并没有太多考虑其他因素比如服务器接口的承载能力以及网页安全问题等，等以后学习了更多之后再找时间重构一下。


## 项目目录


```
└── myblog
    ├──components # 自定义组件
    ├──pages # 页面组件
    ├──actions # 动作封装
    ├── node_modules # 依赖模块
    ├── package.json # 依赖配置
    ├── app.js # 应用
```
